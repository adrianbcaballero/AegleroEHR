# HIPAA Risk Analysis

> **Status:** Not started

Formal Risk Analysis per HIPAA Security Rule §164.308(a)(1)(ii)(A): asset inventory of every system component holding ePHI, threat catalog (insider abuse, lost credentials, ransomware, supply chain, AZ outage, etc.), vulnerability assessment, likelihood × impact ratings, existing controls mapping, residual risk after controls, and treatment decisions (accept/mitigate/transfer/avoid) for every medium-or-high residual.
